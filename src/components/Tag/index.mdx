## API文档
|  参数  |  说明  |  类型  |  默认值  |
|  ---   |  ---  |  ---  |  ---  |
|  closable  |  是否可关闭  |  bool  |  false  |
|  onClose  |  标签关闭时的回调  |  func  |   |
|  color  |  标签的颜色,不设置则为默认颜色  |  string  |  '#fafafa'  |
